# 🇮🇷 فارسی‌نویس هوشمند برای ClientMod

<p align="center">
  <img src="لینک_عکس_اصلی_شما_اینجا" alt="FarsiNevis Preview" width="600">
</p>

دیگر نگران تایپ فارسی در بازی **Counter-Strike: Source** نباشید! این ابزار سبک و سریع، امکان چت فارسی را در کلاینت‌مود برای شما فراهم می‌کند.

---

## 🚀 دریافت و نصب
1. **دانلود:** به بخش **[Releases](لینک_بخش_ریلیز_گیت‌هاب_خود)** بروید و آخرین نسخه `FarsiNevis.exe` را دانلود کنید.
2. **اجرا:** برنامه را اجرا کنید (آیکون آن در System Tray ویندوز ظاهر می‌شود).
3. **آماده‌سازی:** وارد بازی شوید و از کلید میانبر (Hotkey) برای باز کردن پنجره فارسی‌نویس استفاده کنید.

---

## ⚙️ تنظیمات حیاتی (برای عملکرد صحیح)
برای جلوگیری از تداخل و قطع صدای بازی، تنظیمات زیر الزامی است:

* **تنظیمات Launch Options:** در کلاینت‌مود، کد زیر را در تنظیمات بازی وارد کنید:
  ` -windowed -noborder `

* **تنظیم صدای بازی:** برای اینکه هنگام تایپ فارسی صدای بازی قطع نشود، در کنسول بازی (`~`) بزنید:
  ` snd_mute_losefocus 0 `

---

## 📝 نحوه استفاده
1. با کلید میانبر، پنجره برنامه را باز کنید.
2. متن فارسی خود را بنویسید.
3. **ارسال:**
   - فشردن **Enter**: ارسال به چت عمومی (All Chat).
   - فشردن **Shift + Enter**: ارسال به چت تیمی (Team Chat).
   
*برنامه به صورت خودکار متن را کپی کرده، کلید Y یا U را زده و متن را برای شما Paste می‌کند.*
